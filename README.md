\# EuroGrid Balance



A C++ based energy balancing system that simulates energy production, consumption, and energy trade between European countries.



\## Features



\- Energy production and consumption analysis

\- Optimal energy transfer between countries

\- Penalty calculation for energy deficits

\- Multiple report generation

\- Graph-based country energy network



\## Technologies



\- C++

\- File Processing

\- Graph Logic

\- QuickSort Algorithm



\## Files



\- main.cpp -> main program

\- ulkeler.txt -> country data

\- enerjihatlari.txt -> energy transmission distances
